Worldwide Weather

Worldwide Weather is a project that I built to get familiar using APIs and displaying the data in a way that is best for the user experience. The animated weather icons, as well as the coloured backgrounds, give the user more of a visual explanation of the weather.

I also added a feature to locate the current location of the user and display the corresponding data to them in a way that is clean and easy to understand.
